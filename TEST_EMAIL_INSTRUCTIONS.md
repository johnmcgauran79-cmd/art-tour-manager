# Email Testing Instructions

Please test the email functionality now by:

1. Go to a tour with bookings (e.g., 2025 Melbourne Cup Tour)
2. Click "Email All"
3. Select "Default Booking Confirmation" template
4. Click "Send X Emails"

After sending, the edge function logs will show detailed debugging information about:
- Whether custom content is being provided
- How many hotel bookings exist
- The actual hotel data being processed
- Whether templates are being processed correctly

This will help us identify the exact issue with hotel fields not appearing.
